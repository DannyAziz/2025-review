# Danny's Claude Code Plugins

A collection of personal productivity plugins for Claude Code.

## Installation

```
/plugin marketplace add dannyaziz/2025-review
```

## Available Plugins

### 2025-review

An interview-style annual review guide that walks you through:
- Monthly timeline reflection
- 7 aspects of life (Health, Wealth, Mental, Vocational, Family, Social, Spiritual)
- Deep reflection questions
- Desires and intentions for 2026

**Install:**
```
/plugin install 2025-review@danny-skills
```

**Use:**
```
/review
```

## Adding This Marketplace Locally

For local testing:
```
/plugin marketplace add ./path/to/2025-review
```
